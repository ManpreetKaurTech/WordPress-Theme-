Theme Installation & Setup Guide
Step 1: Install the Theme

a.Go to your WordPress installation directory:
wp-content/themes/

b.Upload and extract the my-theme.zip file here.
(Your theme folder should look like wp-content/themes/elearning/)

c.og in to your WordPress Admin Dashboard.
Go to Appearance → Themes and Activate “Elearning”.

Step 2: Create a Primary Menu

a.Go to Appearance → Menus in the Dashboard.

b.Click Create a New Menu and name it Primary Menu.

c.Add your desired pages or custom links.

d.Check the box Display location → Primary Menu and click Save Menu.

Step 3: Edit Footer Text

a.Go to Appearance → Widgets.

b.Locate the Footer Widget Area (or similar name).

c.Edit the footer text according to your requirements.

d.Save the changes and refresh your website to see updates.
